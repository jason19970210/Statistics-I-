## Topics
1. What is __statistics__
- The science of conducting studies to **collect**, **organize**, **summarize**, **analyze**, and draw **conclutions** from data.
2. 解決統計問題的五大步驟：
    1. 問題定義
    2. 資料蒐集
        - 抽樣方法
    3. 資料整理
        - 中央趨勢量數
        - 分散量數
        - 統計圖表
    4. 資料分析
        - 常用統計量
        - 信賴區間
        - 假設檢定
        - 變異數分析(ANOVA)
        - 迴歸分析
    5. 結論與決策
        - 可信度
3. Terminology 專業用語
- Variable 變數
- Data 資料
- Population 母體
- Sample 樣本
